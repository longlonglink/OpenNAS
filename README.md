# OpenNAS

A high-performance, low-power open-source Network Attached Storage (NAS) project.

## Project Overview

OpenNAS is an open-source NAS project aimed at helping users regain sovereignty over their data. In an era where data is growing exponentially, we stand at a critical juncture in Internet history. OpenNAS is not just an ordinary storage device; it is the fundamental infrastructure for individuals in the digital world.
